# Installation

## Development

1. Create .env file.

```bash
cp env.example .env
```

2. Load fake data

``` bash
make run.loaddata.test
```

### Run Gulp

```
gulp dev
```

### Recreate docker

```
make docker.recreate.django
```

### Load fake data

```
make run.loaddata.test
```

## Production

1. Create .env file.

```bash
cp env.example .env
```

2. Load minimum data

``` bash
make run.loaddata
```

3. Create superuser

``` bash
docker-compose exec -T django bash -c "python3 manage.py createsuperuser"
```
